# 🎬 Movies App - Système de Recommandation de Films

## 📌 Présentation du Projet
Ce projet est une **application de recommandation de films** utilisant **l'apprentissage automatique (ML)** pour suggérer des films basés sur les préférences des utilisateurs.  
L'application est développée avec **Python et Streamlit**, et s'appuie sur des **datasets IMDB et TMDB**.

🚀 **Live Demo:** [Cliquez ici](https://app-movie-recommendation-fcnq7fa4kixrswrsxcihpo.streamlit.app/)

---

## 🚀 Fonctionnalités
✅ **Interface interactive Streamlit** pour une navigation fluide  
✅ **Système de recommandation basé sur l'apprentissage automatique**  
✅ **Exploration des films avec filtres personnalisés**  
✅ **Visualisation des tendances des films**
